# EXPENSE_TRACKER

This is a simple and beginner friendly expense tracker built with python. This project helps users record,view, and manage their daily expenses while strengthening core python programming concepts such as file handling, data structure, and user input validation.

# Project Overview

Managing expenses in an essential life. This Expense Tracker allows users to:
- Add ne expenses
- View all recorded expenses
- Track spending habits
- Store data persistently using files

## Features
- Add expenses(amount)
- View all saved expenses
- Persistent storage using .json
- Simple and clean logic for easy understanding
- Console based interface
